# Rag-project 🚀

A Retrieval-Augmented Generation (RAG) based Question-Answer Assistant.

---

## 📌 Overview

This project allows users to upload documents and ask questions. The system retrieves relevant information from the documents and generates intelligent responses using AI models.

---

## ✨ Features

- 📄 Document Processing
- 🔍 Semantic Search
- 🤖 AI-Powered Responses
- ⚡ Fast Retrieval System

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Sentence Transformers
- Vector Embeddings
- RAG Architecture

---

## 🚀 Getting Started

### Installation

```bash
pip install -r requirements.txt
```

### Running the Application

```bash
streamlit run app.py
```
